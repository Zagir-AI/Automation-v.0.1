# ElectroProject — система автоматизации электропроектирования

Инструмент для автоматического расчёта электроснабжения зданий, подбора кабелей и автоматов, проверки ПУЭ, и генерации комплекта проектной документации по ГОСТ.

## Возможности

- **Расчёт нагрузок** — мощность, ток, cosφ, ΔU по ПУЭ и СП 256
- **Подбор кабелей** — марка, сечение, допустимый ток, проверки ΔU / термо-КЗ / чувствительности защиты
- **Подбор автоматов** — серии IEK, Schneider, ABB, DEKraft, TDM
- **КРМ** — компенсация реактивной мощности, автоматический подбор батареи
- **Категорийность** — определение категории здания 1/2/3 по ПУЭ гл.1.2, схема ВРУ (АВР / резервный ввод / одиночный)
- **Проверка селективности** — коэффициент ≥ 1.6 между ступенями защиты
- **Наружные сети** — расчёт освещения по СП 52, ΔU уличных линий
- **DXF/AutoCAD** — синхронизация атрибутов блоков, нумерация кабелей, однолинейная схема, планы
- **Документы по ГОСТ** — спецификация 21.110, кабельный журнал 21.613, ведомость нагрузок, ведомость объёмов работ, программа ПНР
- **Импорт** — парсинг заданий смежников из DXF, таблиц нагрузок Excel/CSV, смет/КП поставщиков
- **Веб-интерфейс** — Streamlit UI с 6 вкладками

## Быстрый старт

```bash
cd elec-system
pip install -r requirements.txt

# Посмотреть демо-объект
python cli.py summary projects/DEMO-2025-001_Офисный_центр

# Пересчитать
python cli.py calc projects/DEMO-2025-001_Офисный_центр

# Сгенерировать все документы
python cli.py docs projects/DEMO-2025-001_Офисный_центр

# Веб-интерфейс
streamlit run ui/app.py
```

```

  projects/                 ← объекты проектирования
    КОД_Название/

  calc/
    engine.py               ← расчётный движок
    compensation.py         ← расчёт КРМ
  docs/
    gen_spec.py             ← спецификация ГОСТ 21.110
    gen_cable_journal.py    ← кабельный журнал ГОСТ 21.613
    gen_load_tables.py      ← ведомость нагрузок
    gen_work_list.py        ← ведомость объёмов работ
    gen_pnr.py              ← программа ПНР
  rules/
    category_rules.py       ← категорийность ПУЭ гл.1.2
    selectivity.py          ← проверка селективности АВ
  data/
    cables/pue_tables.py    ← таблицы допустимых токов ПУЭ
    breakers/               ← ряды номиналов, серии АВ
    demand_factors/         ← коэффициенты спроса СП 256
  parsers/                  ← импорт из DXF/Excel/CSV
  dwg/                      ← генерация и обновление DXF
  outdoor/                  ← расчёт наружных сетей освещения

  changes/                  ← журнал ревизий по ГОСТ 21.1101
```

## CLI-команды

| Команда | Описание |
|---|---|
| `calc <проект>` | Расчёт ВРУ и всех сетей |
| `calc-outdoor <проект>` | Расчёт наружных сетей |
| `summary <проект>` | Сводка: мощности, кабель, АВ, категория здания |
| `check-cables <проект>` | ΔU / термо-КЗ / чувствительность защиты |
| `check-selectivity <проект>` | Проверка селективности АВ |
| `check-compensation <проект>` | Расчёт КРМ |
| `validate <проект>` | Проверка корректности project.json |
| `docs <проект> [--type spec\|cable\|load\|work\|pnr]` | Генерация документов |
| `stamp <проект> [--field … --value …]` | Просмотр/правка штампа |
| `change <проект> --desc … --reason 01-04` | Регистрация ревизии |
| `number-cables <проект>` | Нумерация кабелей КЛ-ЩИТ-NN |
| `plan <проект> [--section ЭОМ]` | Генерация DXF-планов |
| `sld <проект>` | Однолинейная схема DXF |
| `update-attribs <проект> --dxf …` | Синхронизация атрибутов AutoCAD |
| `import <проект> dwg\|table <файл> --section …` | Импорт от смежников |
| `new <КОД> <Название>` | Создать новый объект |
| `list` | Список всех объектов |

## Структура project.json

```
project{}          ← реквизиты: name, code, stage, designer, org, …
vru{}              ← ВРУ → feeders[] → panels[] → consumers[]
building{}         ← параметры здания: floor_height_m, floors
outdoor_networks[] ← наружное освещение
extra_items[]      ← ручные позиции спецификации
changes[]          ← журнал изменений (ревизии)
cable_numbering{}  ← нумерация кабелей {id → "КЛ-ЩИТ-NN"}
_results{}         ← результаты расчёта (не редактировать вручную)
_building{}        ← категория здания, схема ВРУ, compliance
```

Поле `section_mm2: null` означает автоподбор сечения. Число — фиксированное сечение.  
Поле `reserve: true` — кабель и АВ подбираются, в нагрузку потребитель **не** включается.
